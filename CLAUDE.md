# 情報科 授業計画ディレクトリ

高校情報科（情報Ⅰ・情報Ⅱ）の授業計画を年度単位で管理する。

## ディレクトリ構成

```
edu/
├── docs/                   ← すべてのコンテンツはここ以下
│   ├── _template/
│   │   ├── 01_年間計画.md
│   │   ├── 02_評価まとめ.md
│   │   ├── 03_振り返り.md
│   │   ├── 01_単元計画.md
│   │   ├── 02_評価まとめ.md
│   │   └── 授業/授業.md
│   └── YYYY/
│       ├── informa1/
│       │   ├── 01_年間計画.md
│       │   ├── 02_評価まとめ.md
│       │   ├── 03_振り返り.md
│       │   └── NN_単元名/
│       │       ├── 01_単元計画.md
│       │       ├── 02_評価まとめ.md
│       │       └── 授業/
│       │           └── NN_タイトル.md
│       └── informa2/       ← 実施しない年は作らない
├── mkdocs.yml
└── .github/workflows/deploy.yml
```

## 命名規則

- 年度: `2026`, `2027` ...
- 科目: `informa1`, `informa2`
- 単元フォルダ: `01_単元名`（2桁ゼロパディング）
- 授業ファイル: `01_タイトル.md`（2桁ゼロパディング）

## ファイルの役割

| ファイル | 内容 |
|---|---|
| 01_年間計画.md | 科目目標・目指す生徒像・評価基準・評価方法・単元一覧 |
| 02_評価まとめ.md（科目） | 全単元を統合した最終評価 |
| 03_振り返り.md | 年度末の改善メモ・次年度への申し送り |
| 01_単元計画.md | 単元目標・評価基準・評価方法・授業構成・使用教材 |
| 02_評価まとめ.md（単元） | 単元終了後の観点別評価結果・気づき |
| 授業/NN_○○.md | 個別授業の目標・展開・評価 |

## スキル

- `/subject` : 科目ディレクトリと年間計画・評価・振り返りファイルを作成
- `/unit` : 単元ディレクトリと単元計画・授業ファイルを作成
