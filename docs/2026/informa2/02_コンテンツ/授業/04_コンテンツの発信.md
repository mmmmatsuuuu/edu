# 第4時：コンテンツの発信

## 本時の目標

- プル型メディアとプッシュ型メディアの違いを説明し、目的に応じた使い分けを考えられる
- ネイティブアドとターゲティング広告の仕組みを理解し、発信者・受け手それぞれの視点から考察できる
- インプレッション・コンバージョン・KPIの概念を理解し、コンテンツ配信の評価と改善の流れを説明できる

## キーワード

| 用語 | 説明 |
|---|---|
| プル型メディア | ユーザーが能動的に情報を取りに行くメディア。検索エンジンやWebサイトがこれにあたる |
| プッシュ型メディア | 発信者側からユーザーに情報を届けるメディア。メルマガ・プッシュ通知・テレビCMなど |
| インプレッション | コンテンツや広告が画面に表示された回数。どれだけ多くの人の目に触れたかを示す指標 |
| コンバージョン | Webサイトやコンテンツを通じて、購入・登録・クリックなど目標とする行動をユーザーが達成すること |
| KPI（重要業績評価指標） | 目標の達成度を測るための具体的な数値指標。コンバージョン率・クリック率・視聴回数などがこれにあたる |
| ネイティブアド | 周囲のコンテンツと見た目や雰囲気を合わせ、広告と気づかれにくいように設計された広告形式 |
| ターゲティング広告 | ユーザーの属性・行動・興味関心などのデータをもとに、特定のユーザーに絞って配信する広告 |

## 準備物・教材

【教師】
- 授業動画（約12分）：プル型/プッシュ型メディアの違い・ソーシャルメディアによる発信・広告の仕組み（ネイティブアド・ターゲティング広告）・配信評価と改善（KPI・コンバージョン）
- 分析・設計ワークシート（Google Classroom 配布）

【生徒】
- 学習用端末
- 教科書 情報Ⅱ（東京書籍）第2章

## 授業展開

| 時間 | 段階 | 学習活動 | 指導上の留意点 |
|---|---|---|---|
| 5分 | 導入 | 「最近SNSや動画で見た広告を思い出してみよう。それは何で自分に届いたと思うか？」をペアで共有する | 「自分で検索したわけじゃないのに表示された」という経験からターゲティング広告への問いにつなげる。プッシュ型の具体的な感覚を引き出す |
| 20分 | 展開①（動画） | 教師作成動画を視聴。内容：①プル型メディアとプッシュ型メディアの違いと具体例②ソーシャルメディアを使った発信の特徴（拡散・エンゲージメント・アルゴリズム）③広告の仕組み（ネイティブアドの特徴と見分け方・ターゲティング広告のデータ活用）④インプレッションとコンバージョンの違い⑤KPIを設定して評価・改善するサイクル（目標→発信→計測→改善） | 動画約12分。ネイティブアドは「広告であることを明示する義務（ステルスマーケティングの問題）」に触れる。ターゲティング広告はプライバシーとのトレードオフも簡単に言及する |
| 20分 | 展開②（分析・設計活動） | ワークシートに取り組む。①「デジタルスタンプラリーのガチャ景品」を来場者に届けるとしたら、プル型・プッシュ型のどちらが適切か、理由とともに選ぶ②身近なSNS投稿やWeb広告を1つ取り上げ、ネイティブアドかどうかを判断してその根拠を書く③ガチャ景品の配信ページに設定するKPIを考え、どの指標を使えば「成功・失敗」を判断できるかを記述する | ①は「来場者が能動的に検索するか・こちらから通知するか」の観点で考えさせる。②は「広告らしくない広告」の例を事前に1〜2件提示しておく。③はKPIが「目標と連動している」かどうかを確認させる（例：コンバージョン率＝景品を受け取った人÷ページを見た人） |
| 5分 | まとめ | 振り返りフォームに記入：①コンテンツの発信で最も大事だと思った観点②次の演習で自分たちのガチャ景品をどう「届けるか」について考えたことを一言 | 次時（演習①：ガチャ景品企画）では、座学で学んだメディア・UI・発信の知識を統合して企画に落とし込むことを予告する |

## 板書・スライド構成

（動画内のスライド構成）
1. 本時のめあて・キーワード一覧
2. プル型 vs プッシュ型メディアの比較（具体例つき）
3. ソーシャルメディア発信の特徴（拡散・エンゲージメント・アルゴリズムの図解）
4. ネイティブアドとは（通常コンテンツとの見た目の比較・ステルスマーケティングとの違い）
5. ターゲティング広告の仕組み（データ収集→分析→配信の流れ）
6. インプレッション・コンバージョン・KPIの関係図
7. 評価・改善サイクル（目標設定→発信→計測→改善）
8. 本時のまとめ：コンテンツは「届け方」の設計で価値が変わる

## 本時の評価

- **知識・技能**：ワークシートのネイティブアド判断から、広告形式の特性を正確に理解しているか確認
- **思考・判断・表現**：KPI設計の記述から、目標と指標を対応させて根拠ある評価基準を設定できているか評価
