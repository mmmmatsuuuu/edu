# 第1時：メディアの特性と利用

## 本時の目標

- 通信メディアの種類と各特性を説明できる
- オウンドメディア・ペイドメディア・アーンドメディアの違いを理解し、目的に応じた使い分けを考えられる
- メディアミックス・クロスメディアの概念を理解し、具体的な事例で説明できる

## キーワード

| 用語 | 説明 |
|---|---|
| メディア | 情報を記録・伝達するための手段や媒体の総称。通信メディアと表現メディアに大別される |
| マスメディア | テレビ・ラジオ・新聞・雑誌など、不特定多数に対して同時に情報を届ける媒体 |
| インターネットメディア | インターネットを通じて情報を発信・受信するメディア。双方向性・即時性・情報の拡散が特徴 |
| オウンドメディア | 企業や個人が自ら所有・運営するメディア（自社Webサイト・公式ブログなど） |
| ペイドメディア | 費用を支払って利用する広告メディア（Web広告・テレビCM・検索連動広告など） |
| アーンドメディア | SNSの口コミ・第三者によるレビューなど、信頼や評判によって獲得されるメディア |
| メディアミックス | 同一のコンテンツを漫画・アニメ・ゲームなど複数のメディアで展開する戦略 |
| クロスメディア | 複数のメディアを連携させ、受け手が各メディアを横断しながら情報を受け取れるよう設計する手法 |

## 授業展開

| 時間 | 段階 | 学習活動 | 指導上の留意点 |
|---|---|---|---|
| 5分 | 導入 | 「昨日1日で、どんなメディアから情報を受け取ったか」をペアで挙げ、共有する | テレビ・YouTube・SNS・新聞・口コミなど多様な例が出ることを想定し、「それぞれ何が違うのか」という問いにつなげる |
| 20分 | 展開①（動画） | 教師作成動画を視聴。内容：①通信メディアの種類と特性（マスメディア・インターネットメディアの比較）②オウンドメディア/ペイドメディア/アーンドメディアの違いと役割③メディアミックスの事例（人気IPの複数展開）④クロスメディアの事例（テレビ×SNS×Webの連携） | 動画約12分。マスメディアとインターネットメディアは「一方向か双方向か」「即時性があるか」の2軸で対比させる。オウンド/ペイド/アーンドはデジタルスタンプラリープロジェクトとの関連（自分たちはどのメディアで発信するか）を意識させる |
| 20分 | 展開②（分類・分析活動） | ワークシートに取り組む。①「テレビCM・Instagram広告・自社公式サイト・食べログのクチコミ・Xのトレンド入り」をオウンド/ペイド/アーンドに分類し、理由を書く②身近な事例（好きなIP・ブランド・地域イベントなど）を1つ選び、メディアミックスまたはクロスメディアの観点から分析する | ①は「費用を払っているか」「自分でコントロールできるか」の2点を判断基準として示す。②は「なぜそのメディアを組み合わせているのか」の目的や戦略まで考えさせる |
| 5分 | まとめ | 振り返りフォームに記入：①最も印象に残ったメディアの区分とその理由②プロジェクト（デジタルスタンプラリー）では、どのメディアを使って情報を届けるべきかを一言で書く | プロジェクトとの接続を意識させる。次時（コンテンツの制作①）では「何を・どんな形式で作るか」に進むことを予告する |

## 板書・スライド構成

（動画内のスライド構成）
1. 本時のめあて・キーワード一覧
2. 通信メディアの種類と特性（マスメディア vs インターネットメディアの比較表）
3. オウンドメディア・ペイドメディア・アーンドメディアの図解（トリプルメディア）
4. メディアミックスの事例（漫画→アニメ→ゲーム→映画 など）
5. クロスメディアの事例（テレビCM→QRコード→Web→SNS拡散）
6. 本時のまとめ：メディアは「誰が・どんな目的で・どう使うか」で選ぶ

## 本時の評価

- **知識・技能**：ワークシートの分類問題から、オウンド/ペイド/アーンドメディアの区別を正確に理解しているか確認
- **思考・判断・表現**：事例分析の記述から、メディアの特性と目的を結びつけて考察できているか評価
