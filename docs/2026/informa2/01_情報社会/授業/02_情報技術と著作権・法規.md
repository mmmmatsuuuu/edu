# 第2時：情報技術と著作権・法規

## 本時の目標

- AI生成コンテンツが著作権法上どう扱われるかを、日本の現行法と世界の動向を踏まえて説明できる
- CCライセンス・パブリックドメインの読み方を理解し、ガチャ配信で使える素材を判断できる

## キーワード

| 用語 | 説明 |
|---|---|
| 著作物の定義 | 著作権法における著作物の条件。「思想または感情を創作的に表現したもの」とされ、AIが自律的に生成したコンテンツは現行法では原則として著作物に該当しない |
| AI生成コンテンツと著作権 | AIが生成した画像・文章などの権利の扱い。日本では人の創作性が介在した場合のみ著作権が発生しうる |
| 著作権法47条の5 | AIの機械学習のためのデータ収集・複製は一定の範囲で認められている。ただし生成物の商業利用には別途注意が必要 |
| クリエイティブ・コモンズ（CC） | 著作権者が利用条件を事前に公示する仕組み。条件に従えば許可なく利用できる |
| パブリックドメイン | 著作権が消滅または放棄された作品。誰でも自由に利用できる |
| 個人情報保護法 | 個人情報の収集・利用・管理のルールを定めた日本の法律 |

| 20分 | 展開②（法的判断演習） | ワークシートに取り組む。①「AIで生成した画像をガチャ景品にする場合にどんな問題が起こりうるか」を考え記述②CCライセンス記号の読み取り問題（画像を配信してよいか判断）③「プロンプトに自分の創意工夫を加えた場合、著作権は発生するか」について自分の考えを書く | 日本の現行法の解釈（原則：AI単独生成は著作物にならない）と、世界での議論がまだ続いていることを踏まえて書かせる。「わからない」「グレーゾーン」も正直に書いてよいと伝える |
| 5分 | まとめ | 振り返りフォームに記入：①AI生成コンテンツを使う際に自分が気をつけたいこと②CCライセンスで特に重要と感じた条件 | 演習①（第6時）で著作権確認シートを作成することを予告し、本時の学びが直結することを意識させる |
