# 第2時：情報技術と法律・ガバナンス

## 本時の目標

- 著作権（財産権）・CCライセンス・パブリックドメインの概要を理解し、コンテンツ配信における実務的な判断ができる
- 個人情報保護法・GDPR・AIガバナンスの必要性と限界について自分の考えを持てる

## キーワード

| 用語 | 説明 |
|---|---|
| 著作権（財産権） | 著作物を経済的に利用する権利。複製権・公衆送信権・譲渡権などからなり、譲渡が可能 |
| パブリックドメイン | 著作権が消滅または放棄された作品。誰でも自由に利用できる |
| クリエイティブ・コモンズ（CC） | 著作権者が利用条件を事前に公示する仕組み。条件に従えば許可なく利用できる |
| GDPR | EUの一般データ保護規則。個人データの収集・利用に関する厳格なルールを定める |
| AIガバナンス | AIの開発・利用を適正に管理するための方針・規制・体制のこと |
| 個人情報保護法 | 個人情報の収集・利用・管理のルールを定めた日本の法律 |

## 授業展開

| 時間 | 段階 | 学習活動 | 指導上の留意点 |
|---|---|---|---|
| 5分 | 導入 | 「ネットで拾った画像をそのままSNSに投稿するとどうなる？」を問いかけ、著作権への気づきを促す | 情報Ⅰで学んだ著作権の知識を発展させる出発点として位置づける。「知らなかった」では済まないことを実感させる |
| 20分 | 展開①（動画） | 教師作成動画を視聴。内容：①著作権（財産権）の種類（複製権・公衆送信権・譲渡権等）②CCライセンスの6種類と読み方③パブリックドメインの定義と素材の探し方④個人情報保護法の改正ポイント（2022年）⑤GDPR の概要⑥AIガバナンスの動向（EU AI法・日本のガイドライン） | 動画約12分。CCライセンスは「BY（表示）・NC（非営利）・SA（継承）・ND（改変禁止）」の記号の意味を特に丁寧に解説する |
| 20分 | 展開②（実務的判断演習） | ワークシートに取り組む。①CCライセンス記号の読み取り問題（画像を配信してよいか判断）②「ガチャで配信するとしたらどんな素材が使えるか」をCC・パブリックドメインの観点で考える③個人情報保護法とGDPRの違いを表で整理 | 「ガチャ配信」という具体的な文脈でCC・パブリックドメインを考えさせる。後の演習①（第6時）につながる準備であることを伝える |
| 5分 | まとめ | 振り返りフォームに記入：①CCライセンスで気をつけるべきこと②法律で規制すべきことと自分たちのリテラシーで対応すべきことの境界線はどこか | 法規制と個人リテラシーの両輪という観点を意識させる |
