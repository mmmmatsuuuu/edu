# 第3時：情報の信頼性とメディアリテラシー

## 本時の目標

- 情報の信頼性を評価する手法（ファクトチェック・SIFT法等）を理解できる
- フィルターバブル・エコーチェンバーのメカニズムを説明し、対策を考えられる

## キーワード

| 用語 | 説明 |
|---|---|
| メディアリテラシー | メディアの特性を理解し、情報を批判的に読み解く能力 |
| SIFT法 | 情報の信頼性を評価する手順。Stop（立ち止まる）・Investigate（発信者を調べる）・Find better sources（より良い情報源を探す）・Trace claims（根拠をたどる）の頭文字 |
| ファクトチェック | 情報の事実関係を検証し、正確性を確かめる行為・仕組み |
| フィルターバブル | アルゴリズムが利用者の好みに合った情報だけを表示し、偏った情報環境が生まれる現象 |
| エコーチェンバー | 同じ意見の人だけが集まる環境で、特定の考えが増幅・強化される現象 |

## 授業展開

| 時間 | 段階 | 学習活動 | 指導上の留意点 |
|---|---|---|---|
| 5分 | 導入 | 「SNSで見た情報をそのままシェアしたことはあるか？その情報は正しかったか？」を問いかける | 経験から入ることで当事者意識を高める。責めるのではなく、誰でも誤情報を広めうることを伝える |
| 20分 | 展開①（動画） | 教師作成動画を視聴。内容：①情報の信頼性を評価する観点（発信者・根拠・日時・一次情報）②SIFT法（Stop・Investigate・Find better sources・Trace claims）③ファクトチェックサイトの活用④フィルターバブルとは（アルゴリズムによる情報の偏り）⑤エコーチェンバーとは（同質意見だけに囲まれる状態） | 動画約12分。情報Ⅰで学んだフェイクニュースの知識を発展させる形で提示する |
| 20分 | 展開②（記事検証） | ワークシートの事例（SNS投稿・ニュース記事）に対してSIFT法を使って信頼性を検証。①発信者は誰か②根拠・出典はあるか③一次情報に当たれるか④ファクトチェックサイトで確認⑤総合的な信頼性の評価と理由 | 実際にブラウザで検索しながら進めてよいことを伝える。答えより「調べ方・判断の根拠」を重視する |
| 5分 | まとめ | 振り返りフォームに記入：①情報の信頼性を確かめるためにこれから使う方法②フィルターバブルから抜け出すために自分ができること | 実生活への適用を意識させる |
