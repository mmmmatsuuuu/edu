# 第6時：演習①「著作権確認と素材制作」

## 本時の目標

- ガチャで配信するコンテンツに使う素材の著作権ライセンスを調査・確認できる
- ライセンスのルールに沿った素材を実際に制作または収集し、利用可能性を判断できる

## キーワード

| 用語 | 説明 |
|---|---|
| ライセンス | 著作物を利用する際の条件・許可の仕組み。どこまで自由に使えるかを定めたルール |
| 二次利用 | 既存の著作物をもとに別の用途・形式で使用すること。ライセンスの確認が必須 |
| 帰属表示（クレジット） | CCライセンスなどで素材を使う際に必要な、作者名・出典の明記 |
| 自作素材 | 著作権上の問題を回避する最も確実な手段。一から自分で制作した素材 |

## 準備物・教材

【教師】
- 演習説明スライド（活動の流れ・CCライセンスの読み方復習・素材サイト一覧）
- 著作権確認シート（Google Classroom 配布、Google Docs）
- フリー素材サイト一覧（Unsplash・Pixabay・いらすとや・Public Domain Q等）

【生徒】
- 学習用端末
- Google アカウント（素材サイトへのアクセス・Canva等）
- 著作権確認シート

## 著作権確認シートの構成

```
【素材のURL・出典】：
【ライセンス種別】：（例：CC BY 4.0 / パブリックドメイン / 自作 等）
【商用利用可否】：可 / 不可 / 要確認
【改変可否】：可 / 不可 / 要確認（SA条件あり/なし）
【帰属表示の要否と方法】：
【ガチャ配信への利用可否の判断】：可 / 不可
【理由・メモ】：
```

## 授業展開

| 時間 | 段階 | 学習活動 | 指導上の留意点 |
|---|---|---|---|
| 10分 | 導入・説明 | 演習の目的・手順・評価基準を説明。CCライセンスの読み方を簡単に復習。使用できる素材サイト一覧と確認シートの記入方法を実演 | 「ガチャ配信 = 不特定多数への公衆送信」という法的文脈を意識させる。商用利用・改変・帰属表示の3点が特に重要と伝える |
| 30分 | 作業 | ①フリー素材サイトから候補素材を3点以上探し、著作権確認シートに記録する②「利用できない」と判断した場合は理由を明記③自作素材を1点以上制作し（Canva・手書きのデジタル化等）、確認シートに記録する | 複数のCCライセンス種別が混在するよう、素材の幅を広く探させる。「自作すれば著作権問題は出ないが、著作者人格権は残る」という点も確認させる |
| 10分 | まとめ | 確認シートを提出。気づいたことをシートのメモ欄に記入。次時（開発計画の策定）の概要を説明 | 「法的に使える素材だけを使う」という姿勢をプロジェクト全体に通じる原則として定着させる |

## 本時の評価

- **知識・技能**：確認シートの記録から、CCライセンスの読み方・パブリックドメインの判断が正確にできているか確認
- **思考・判断・表現**：「利用可否の判断」と「理由」の記述から、法的知識を実務に応用できているか評価
- **主体的に学習に取り組む態度**：自作素材の制作・複数素材の調査への積極的な取り組み

## 対応する学習指導要領の内容

- 情報Ⅱ（1）イ：情報に関する法規・倫理（著作権・CCライセンス・知的財産の実務的活用）
