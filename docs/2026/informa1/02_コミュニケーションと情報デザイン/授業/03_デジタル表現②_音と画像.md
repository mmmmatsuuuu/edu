# 第3時：デジタル表現②：音と画像

## 本時の目標

- 標本化・量子化・符号化の3ステップで音がデジタル化される仕組みを説明できる
- サンプリング周波数・量子化ビット数が音質とデータ量に与える影響を理解できる
- 画素・解像度・階調の概念を理解し、画像のデジタル表現の仕組みを説明できる

## キーワード

| 用語 | 説明 |
|---|---|
| 標本化（サンプリング） | アナログ信号を一定の時間間隔で取り出してデジタル変換する第1のステップ |
| 量子化 | 標本化で取り出した値を決まった段階数に丸める第2のステップ |
| 符号化 | 量子化した値を0と1のデジタルデータに変換する第3のステップ |
| サンプリング周波数 | 1秒間に標本化する回数（Hz）。値が高いほど元の音に近くなるが、データ量も増える |
| 量子化ビット数 | 音の大きさを何段階で表すかを決める値。ビット数が大きいほど細かく・データ量も増える |
| 光の三原色 | テレビやスマホ画面などの基本色であり、R(赤), G(緑), B(青)の3色で構成される |
| 色の三原色 | 絵の具、印刷インクの基本色であり、C(シアン), M(マゼンタ), Y(イエロー)の3色で構成される |
| 画素（ピクセル） | デジタル画像を構成する最小単位の点。各画素が色情報を持つ |
| 解像度 | 画像の縦横に並ぶ画素の数。高いほど鮮明だがデータ量も大きくなる |
| 階調 | 色や明暗を何段階で表現するかの数。ビット深度が大きいほど階調が豊かになる |

## 板書・スライド構成

（動画内のスライド構成）
1. 本時のめあて・キーワード一覧
2. 標本化・量子化・符号化の3ステップ（図解）
3. サンプリング周波数の違いによる音の再現性の比較
4. 量子化ビット数の違いによる音の細かさの比較
5. 画像の構成（画素・解像度の図解）
6. 階調とビット深度の関係（1ビット=2階調〜8ビット=256階調）
7. RGBによる色の表現（24ビットフルカラー）
8. 本時のまとめ：音と画像に共通するデジタル化の仕組み
