# 第1時：コミュニケーションの形態とメディア

## 本時の目標

- コミュニケーションの形態（直接・間接、同期型・非同期型）を理解し、具体的な場面で分類できる
- メディアの種類と特性を理解し、目的に応じた使い分けを考えられる
- ソーシャルメディアがコミュニケーションに与える影響を多角的に考察できる

## キーワード

| 用語 | 説明 |
|---|---|
| メディア | 情報を記録・伝達するための手段や媒体。テキスト・音声・画像・動画など様々な種類がある |
| 直接コミュニケーション | 同じ場所にいる相手と、声・表情・身振りなどを通じて行うコミュニケーション |
| 間接コミュニケーション | 手紙・電話・SNSなど、何らかのメディアを介して行うコミュニケーション |
| 同期型 | 送り手と受け手がリアルタイムでやりとりするコミュニケーションの形態。電話・ビデオ会議など |
| 非同期型 | 送り手と受け手が異なるタイミングでやりとりするコミュニケーションの形態。メール・SNSのメッセージなど |
| ソーシャルメディア | 利用者が情報を発信・共有・拡散できる双方向のデジタルメディア。SNS・動画プラットフォームなどが含まれる |
| メディアリテラシー | メディアの特性を理解し、情報を批判的に読み解き、適切に活用・発信する能力 |

## 授業展開

| 時間 | 段階 | 学習活動 | 指導上の留意点 |
|---|---|---|---|
| 5分 | 導入 | 「今日1日で使ったコミュニケーション手段を全部挙げよう」を問いかけ、ペアで共有する | LINEや対面など身近な手段から入り、コミュニケーションの多様性を実感させる |
| 20分 | 展開①（動画） | 教師作成動画を視聴。内容：①コミュニケーションの構成要素（送り手・受け手・チャネル）②直接・間接コミュニケーションの違い③同期型・非同期型の分類④メディアの種類と特性⑤ソーシャルメディアの特徴と影響（情報拡散・炎上・フィルターバブル）⑥メディアリテラシーとは | 動画約12分。情報Ⅰ単元1で学んだデジタル情報の特性（複製性・伝播性）とつなげて理解させる |
| 20分 | 展開②（分類・考察活動） | ワークシートに取り組む。①通信手段を「直接/間接」「同期型/非同期型」の2軸マトリクスに分類②「友人への謝罪」「商品のクレーム」「イベントの告知」に最適なメディアを選び根拠を書く③「ソーシャルメディアのメリット・デメリット」をそれぞれ考える | 「正解」より選択の根拠を重視する。目的・相手・状況によって最適なメディアが変わることに気づかせる |
| 5分 | まとめ | 振り返りフォームに記入：①本時で気づいたこと②自分のコミュニケーションで見直したいこと | 次時（デジタル表現）への橋渡しとして「情報はどうやってデジタルで伝わるのか」を問いかけて終わる |
