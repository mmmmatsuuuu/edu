# 第4時：情報セキュリティと関連法規

## 本時の目標

- 情報セキュリティの概念（機密性・完全性・可用性）を理解する
- マルウェア・フィッシングなど代表的なサイバー脅威の手口を説明できる
- 知識認証・所有物認証・生体認証の違いを理解し、二段階認証の意義を説明できる
- 不正アクセス禁止法など関連法規の基本を理解する

## キーワード

| 用語 | 説明 |
|---|---|
| 情報セキュリティ | 情報の機密性・完全性・可用性を保護すること。不正アクセスや情報漏洩を防ぐための対策全般を指す |
| マルウェア | コンピュータに害を与えることを目的として作られた悪意あるソフトウェアの総称。ウイルス・スパイウェアなどを含む |
| フィッシング | 銀行や公的機関を装った偽のメール・サイトで、パスワードや個人情報を騙し取る手口 |
| 不正アクセス禁止法 | 他人のIDやパスワードを無断で使い、ネットワークに不正に侵入する行為を禁止した法律 |
| 知識認証 | 本人だけが知っている情報（ユーザーIDとパスワードなど）で本人確認を行う認証方式 |
| 所有物認証 | 本人だけが持っているもの（ICカード・スマートフォンへの通知コードなど）で本人確認を行う認証方式 |
| 生体認証 | 指紋・顔・虹彩など、本人の身体的特徴で本人確認を行う認証方式 |
| 二段階認証 | 知識認証と所有物認証など、異なる種類の認証を2つ組み合わせてセキュリティを高める方式 |
| サイバー犯罪 | インターネットやコンピュータを利用した犯罪行為の総称。不正アクセス・詐欺・個人情報の窃取などを含む |

## 授業展開

| 時間 | 段階 | 学習活動 | 指導上の留意点 |
|---|---|---|---|
| 5分 | 導入 | 「パスワードを使い回したことがあるか」「フィッシングメールを見たことがあるか」を問いかける | 身近な経験をもとに、セキュリティリスクを自分事として捉えさせる |
| 20分 | 展開①（動画） | 教師作成動画を視聴しながらメモを取る。内容：①情報セキュリティの3要素（機密性・完全性・可用性）②マルウェアの種類と手口③フィッシングの仕組み④3種類の認証方式と二段階認証⑤不正アクセス禁止法の概要 | 動画は約12分。残り8分でキーワードを確認しながらメモを整理させる |
| 20分 | 展開②（確認活動） | ワークシートに取り組む。①認証方式を3種類に分類する問題②フィッシングメールの事例を読んで見分け方を考える③「パスワード使い回しのリスク」について説明する | フィッシングの見分け方（URL確認・送信者確認など）を具体的に示す。二段階認証は「知っている＋持っている」の組み合わせが基本と整理させる |
| 5分 | まとめ | 本時の振り返りを Google Forms で記入：①理解できた点 ②疑問点 | 自分のパスワード管理を見直すきっかけにするよう促す |
