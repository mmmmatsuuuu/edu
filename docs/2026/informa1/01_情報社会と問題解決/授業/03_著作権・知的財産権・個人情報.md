# 第3時：著作権・知的財産権・個人情報

## 本時の目標

- 知的財産権の体系（著作権・産業財産権）を理解し、それぞれの権利の種類と内容を説明できる
- 著作者人格権・著作権（財産権）・著作隣接権の違いを理解する
- 個人情報・要配慮個人情報・匿名加工情報の概念を理解し、個人情報保護法の基本的なルールを説明できる

## キーワード

| 用語 | 説明 |
|---|---|
| 知的財産権 | 人の知的な創造活動から生まれた成果を保護する権利の総称。著作権と産業財産権などに分類される |
| 著作権 | 文章・音楽・映像などの創作物を、作った人が独占的に利用できる権利 |
| 著作者人格権 | 著作者の人格的利益を守る権利。公表権・氏名表示権・同一性保持権の3つからなり、譲渡できない |
| 公表権 | 著作物を公表するかどうか、いつ・どのように公表するかを決める権利 |
| 氏名表示権 | 著作物を公表する際に、氏名（実名・ペンネーム・匿名）を表示するかどうかを決める権利 |
| 同一性保持権 | 著作物の内容・題名を、著作者の意に反して改変されない権利 |
| 著作権（財産権） | 著作物を経済的に利用する権利。複製権・上演権・演奏権・上映権・公衆送信権・口述権・展示権・頒布権・譲渡権・貸与権などからなり、譲渡が可能 |
| 複製権 | 著作物をコピー・印刷・録音・録画などで複製する権利 |
| 公衆送信権 | 著作物をインターネット・放送・有線通信で不特定多数に送信する権利 |
| 著作隣接権 | 著作物の創作者ではなく、実演家・レコード製作者・放送事業者など著作物の伝達に貢献した者に認められる権利 |
| 産業財産権 | 特許権・実用新案権・意匠権・商標権の4つからなる、産業上の創作や標識を保護する権利群。特許庁への出願・登録が必要 |
| 特許権 | 新しい発明（技術的アイデア）を保護する権利。特許庁への出願・登録が必要で、最長20年間保護される |
| 実用新案権 | 物の形状・構造・組み合わせに関する考案を保護する権利。特許より保護期間が短い（最長10年） |
| 意匠権 | 製品のデザイン（形状・模様・色彩など視覚的な美しさ）を保護する権利 |
| 商標権 | 商品・サービスに使うブランド名やロゴマークを保護する権利。登録で最長10年、更新で継続できる |
| 引用 | 他者の著作物の一部を自分の文章中に使用すること。出所を明示し、主従関係が明確であれば許可不要 |
| 肖像権 | 自分の顔や姿を無断で撮影・公開されない権利 |
| 個人情報 | 氏名・住所・電話番号・顔写真など、特定の個人を識別できる情報 |
| 要配慮個人情報 | 人種・信条・病歴・犯罪歴など、不当な差別や偏見が生じるおそれがある特に慎重な取り扱いが必要な個人情報 |
| 匿名加工情報 | 特定の個人を識別できないよう加工した個人情報。個人情報保護法の規定に従い、本人の同意なく活用できる |
| 個人情報保護法 | 個人情報の収集・利用・管理に関するルールを定めた法律。事業者に適切な取り扱いを義務づける |
| クリエイティブ・コモンズ（CC） | 著作権者が作品の利用条件をあらかじめ明示できる仕組み。条件を守れば許可なく利用できる |

## 授業展開

| 時間 | 段階 | 学習活動 | 指導上の留意点 |
|---|---|---|---|
| 5分 | 導入 | 「好きなアーティストの曲をSNSで使ったことはあるか」を問いかけ、著作権への関心を引き出す | 具体的な動画投稿・BGM利用の事例を挙げ、身近な問題として捉えさせる |
| 20分 | 展開①（動画） | 教師作成動画を視聴しながらメモを取る。内容：①知的財産権の体系（著作権・産業財産権）②著作者人格権と著作権（財産権）の違い③著作隣接権④個人情報保護法の基本と個人情報の種類 | 動画は約12分。権利の体系図を手元に置いてメモを整理させる。産業財産権4種は「出願・登録が必要」という共通点を強調する |
| 20分 | 展開②（確認活動） | ワークシートに取り組む。①知的財産権の体系図を空欄補充②事例を読んで「引用」として適切か判断・理由を記述③個人情報に該当するかどうかを判定する問題 | 著作者人格権は「一身専属」（本人のみが持つ、譲渡不可）という概念を丁寧に説明する。匿名加工情報の活用例としてビッグデータ分析を紹介する |
| 5分 | まとめ | 本時の振り返りを Google Forms で記入：①理解できた点 ②疑問点 | 内容が多いため「特に重要な概念」を板書で整理してから振り返らせる |

## 板書・スライド構成

（動画内のスライド構成）
1. 本時のめあて・キーワード一覧
2. 知的財産権の体系図（著作権 vs 産業財産権）
3. 著作者人格権の3つの権利（公表権・氏名表示権・同一性保持権）
4. 著作権（財産権）の主な権利一覧（複製権・公衆送信権など）
5. 著作隣接権とは何か（実演家・レコード製作者・放送事業者）
6. 産業財産権4種の比較表（特許権・実用新案権・意匠権・商標権）
7. 引用のルール（主従関係・出所明示）
8. 個人情報の種類（一般・要配慮・匿名加工）
9. クリエイティブ・コモンズの仕組み
10. 本時のまとめ

## 本時の評価

- **知識・技能**：ワークシートから、知的財産権の体系と著作権の種類・個人情報保護法の基本を理解しているか確認
- **思考・判断・表現**：引用の適否を判断する問題で、根拠を挙げて説明できているか
- **主体的に学習に取り組む態度**：振り返りフォームへの記述から学習への姿勢を確認
