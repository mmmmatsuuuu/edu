# 第2時：情報モラルと情報倫理

## 本時の目標

- 情報モラルと情報倫理の違いを理解し、それぞれの概念を説明できる
- SNS上で起こる誹謗中傷・フェイクニュース・炎上の仕組みを理解する
- デジタル社会において責任ある行動（デジタル市民性）をとるための判断基準を考察できる

## キーワード

| 用語 | 説明 |
|---|---|
| 情報モラル | 情報を扱う上で守るべきルールやマナー。他者を傷つけない・プライバシーを尊重するなどの行動規範を含む |
| 情報倫理 | 情報社会における善悪の判断基準や価値観。法律では定められていない行動の正しさを考える領域 |
| SNS（ソーシャルネットワーキングサービス） | 人と人がオンラインでつながり、情報を共有・発信できるサービス。Twitter・Instagram・TikTokなどが代表例 |
| 誹謗中傷 | 根拠のない悪口や侮辱を、特定の人に向けて発信すること。インターネット上での拡散により深刻な被害をもたらす |
| フェイクニュース | 事実ではない情報を、本物のニュースのように見せかけて拡散されるコンテンツ |
| 炎上 | SNSなどで特定の発言・行動が批判を集め、大量の否定的なコメントが殺到する現象 |
| デジタル市民性 | デジタル社会において責任ある行動をとる力。オンライン上のマナー・安全・倫理的判断などを含む |

## 授業展開

| 時間 | 段階 | 学習活動 | 指導上の留意点 |
|---|---|---|---|
| 5分 | 導入 | 「SNSで見かけたトラブルや炎上事例」を思い出してペアで共有する | 実名や特定の事件は避け、一般的な事例を例示しながら関心を引き出す |
| 20分 | 展開①（動画） | 教師作成動画を視聴しながらメモを取る。内容：①情報モラルと情報倫理の違い②誹謗中傷・フェイクニュース・炎上のメカニズム③デジタル市民性とは何か | 動画は約12分。残り8分でキーワードを確認しながらメモを整理させる |
| 20分 | 展開②（確認活動） | ワークシートに取り組む。①情報モラルと情報倫理の違いを自分の言葉で説明②SNSの事例を読んでどのような問題があるか分析③「自分ならどう行動するか」を記述 | 法律の話と倫理の話を意識的に区別させる。「違法ではないが問題のある行動」の例を問いかける |
| 5分 | まとめ | 本時の振り返りを Google Forms で記入：①理解できた点 ②疑問点 | 疑問点は次時の導入で取り上げる |
