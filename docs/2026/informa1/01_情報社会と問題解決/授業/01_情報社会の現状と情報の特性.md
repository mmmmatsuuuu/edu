# 第1時：情報社会の現状と情報の特性

## 本時の目標

- 現代の情報社会の特徴（デジタル化・情報量の急増）を理解する
- 情報の4つの特性（複製性・伝播性・残存性・非物質性）を説明できる
- 情報の特性が日常生活やトラブルとどのように結びついているかを考察できる

## キーワード

| 用語 | 説明 |
|---|---|
| 情報社会 | 情報やデータが社会・経済の中心的な役割を担う社会。デジタル化・ネットワーク化の進展により急速に拡大している |
| デジタル化 | 文字・音声・画像などのアナログ情報を、コンピュータで扱える0と1の数値データに変換すること |
| 複製性 | デジタル情報は劣化なく何度でも複製できるという特性。著作権侵害や情報漏洩の温床になりやすい |
| 伝播性 | 情報はインターネットを通じて瞬時に世界中へ広がるという特性。デマや誹謗中傷が短時間で拡散する要因でもある |
| 残存性 | 一度ネット上に出た情報は半永久的に残り続けるという特性。「デジタルタトゥー」とも呼ばれ、削除しても完全には消えないことが多い |
| 非物質性 | 情報は形や実体を持たないという特性。物と違って同時に複数の人が利用でき、また受け取る人によって解釈が異なるという側面も持つ |

## 授業展開

| 時間 | 段階 | 学習活動 | 指導上の留意点 |
|---|---|---|---|
| 5分 | 導入 | 「1日にどれくらいの情報に触れているか」を考えてペアで話し合う | SNS・動画・ニュースなどを例に挙げ、情報量の多さを実感させる |

| 20分 | 展開②（確認活動） | ワークシートに取り組む。①4つの特性を自分の言葉で説明②「デジタル情報とアナログ情報の違い」を表にまとめる③事例（写真のSNS投稿）を読んで、関係する特性を選択・理由を記述 | 正解を急がせず教科書を参照してよいことを伝える。「同じ情報でも受け取り方が違う」例を問いかけ、非物質性の解釈の多様性に気づかせる |

## 板書・スライド構成

（動画内のスライド構成）
1. 本時のめあて・キーワード一覧
2. 情報社会の変遷（年表形式：アナログ→デジタル→ネットワーク社会）
3. 現代の情報量：「1日に流通する情報量はどのくらい？」
4. 情報の4つの特性（図解）
   - 複製性：コピーしても劣化しない
   - 伝播性：瞬時に世界中へ広がる
   - 残存性：消しても残り続ける
   - 非物質性：形がない・解釈は人それぞれ
5. 特性ごとの事例紹介（SNS投稿・拡散・炎上）
6. 本時のまとめ

## 本時の評価

- **知識・技能**：ワークシートの記述から、情報の4つの特性（複製性・伝播性・残存性・非物質性）を理解しているか確認
- **思考・判断・表現**：事例問題で複数の特性を関連付けて説明できているか
- **主体的に学習に取り組む態度**：振り返りフォームへの記述から学習への姿勢を確認
