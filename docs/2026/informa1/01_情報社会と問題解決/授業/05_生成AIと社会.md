# 第5時：生成AIと社会

## 本時の目標

- 生成AIの仕組みと代表的なサービスを理解する
- ハルシネーション・ディープフェイクなど生成AIが引き起こすリスクを説明できる
- AIリテラシーの観点から、生成AIを適切に活用するための判断基準を考察できる

## キーワード

| 用語 | 説明 |
|---|---|
| 生成AI | テキスト・画像・音声などの新しいコンテンツを自動で生成できるAI（人工知能）。ChatGPT・Geminiなどが代表例 |
| 機械学習 | 大量のデータからパターンを学習し、予測や判断を行う技術。AIの中核をなす手法 |
| ハルシネーション | AIが事実ではない情報を、もっともらしく生成してしまう現象。AIの出力を無批判に信頼しないことが重要 |
| プロンプト | AIに対して与える指示や質問のこと。適切なプロンプトを書くことでAIの出力の質が大きく変わる |
| ディープフェイク | AIを用いて人物の顔や声を合成・改ざんした偽の動画・音声コンテンツ |
| AIリテラシー | AIの仕組みや特性を理解し、適切に活用・批判的に判断できる能力 |

## 授業展開

| 時間 | 段階 | 学習活動 | 指導上の留意点 |
|---|---|---|---|
| 5分 | 導入 | 「ChatGPTやGeminiを使ったことがあるか」を問いかけ、生成AIへの関心と経験を共有する | 使ったことがある生徒の経験を引き出しながら、「便利さ」と「不安」の両面を確認する |
| 20分 | 展開①（動画） | 教師作成動画を視聴しながらメモを取る。内容：①生成AIの仕組み（機械学習・大規模言語モデル）②活用事例③ハルシネーションの仕組みとリスク④ディープフェイクと社会への影響⑤生成AIと著作権の問題⑥AIリテラシーとは | 動画は約12分。「すごい技術だが万能ではない」という視点を意識させる |
| 20分 | 展開②（確認活動） | ワークシートに取り組む。①ハルシネーションの事例を読んでリスクを考える②生成AIを使う際の「やってよいこと・注意すること」を整理③「ディープフェイクが社会に与える影響」についての意見を記述 | 著作権との関係（学習データと引用の問題）は難しいため、「まだ法整備が追いついていない課題」として扱う |
| 5分 | まとめ | 本時の振り返りを Google Forms で記入：①理解できた点 ②疑問点 | 「AIを使うことの責任」について考えるよう促す |
