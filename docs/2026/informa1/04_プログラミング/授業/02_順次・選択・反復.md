# 第2時：順次・選択・反復

## 本時の目標

プログラムの3つの基本構造（順次・選択・反復）を理解し、Pythonの基本構文（if / for / while）と対応づけられるようになる。短いコードを見て「何が出力されるか」を予測（Predict）できる。

## キーワード

| 用語 | 説明 |
|---|---|
| 順次構造 | 上から下へ命令を順番に実行する構造 |
| 選択構造 | 条件によって実行する処理を分ける構造（if / else） |
| 反復構造 | 条件が成り立つ間、処理を繰り返す構造（for / while） |
| 構文 | プログラミング言語における命令の書き方の決まり（インデント・コロンなど） |
| Predict | コードを実行する前に「何が起こるか」を予想する活動 |

## 準備物・教材

- 教科書：情報Ⅰ Flex 第4章
- 授業動画（3構造とif/for/whileの基本構文）※展開①／欠席者対応
- 視聴用ワークシート（構文・3構造の対応表の空欄）
- スライド（フローチャートとコードの対応図）
- 短いサンプルコード集（Predict→Run用。実行環境のリンク付き）

## 授業展開

| 時間 | 段階 | 学習活動 | 指導上の留意点 |
|---|---|---|---|
| 8分 | 導入 | 前回のフローチャートに現れた3構造を振り返り、それぞれがコードのどの形に対応するかを概観する | フローチャートの図形とコードを並べて提示し、対応を意識させる |
| 20分 | 展開①（動画視聴） | 授業動画を視聴し、順次・選択・反復の3構造とPythonの基本構文（if / for / while）、インデント・コロンが構造を決めることを学ぶ。短いコード例の動きも動画で確認する | 視聴中はワークシートの構文・対応表の空欄を埋めながら見るよう指示する |
| 15分 | 展開②（Predict→Run） | サンプルコード数本について、「このコードは何を出力する?」をペアで予想（Predict）→実行して確認（Run）する。予想と違った箇所をワークシートにメモする | 予想を必ず表明させてから実行させる。「なぜ予想と違ったか」を言語化させる。エラーも実行して見せ、構文ミスの例として扱う |
| 7分 | まとめ | 3構造とif/for/whileの対応を板書で整理する。次回の変数・配列が「データを覚える・並べる」役割であることを予告する | 予想が外れた箇所こそ理解が深まることを伝え、次回につなぐ |

## 板書・スライド構成

1. 今日の目標
2. 3構造とコードの対応（フローチャート↔if/for/while）
3. 選択：if / else の構文（条件・コロン・インデント）
4. 反復：for / while の構文
5. Predict→Run の進め方（予想してから実行）
6. まとめ：3構造の整理と変数・配列の予告
