# 第3時：変数と配列

## 本時の目標

変数の概念（値を入れる・書き換える）と配列の構造（複数の値を並べてインデックスで扱う）を理解する。トレース可視化ツールで値の変化を追い、コードの動きを説明できるようになる。

## キーワード

| 用語 | 説明 |
|---|---|
| 変数 | 値に名前をつけて記憶しておく入れもの。代入で値を入れ替えられる |
| 代入 | 変数に値を格納する操作（`x = 5`）。右辺を評価して左辺に入れる |
| 配列（リスト） | 複数の値を順番に並べて1つの名前で扱うデータ構造 |
| インデックス | 配列の各要素の位置を示す番号（0から始まる） |
| トレース | コードを1行ずつ追い、変数の値がどう変化するかを確認すること |

## 準備物・教材

- 教科書：情報Ⅰ Flex 第4章
- 授業動画（変数の概念・配列の構造とインデックス・トレース可視化）※展開①／欠席者対応
- 視聴用ワークシート（空欄・トレース欄）
- トレース可視化ツール（Python Tutor 等）
- サンプルコード（変数の入れ替え・配列の要素アクセス。Predict→Run用）

## 授業展開

| 時間 | 段階 | 学習活動 | 指導上の留意点 |
|---|---|---|---|
| 8分 | 導入 | 「値を覚えておきたい」「たくさんの値をまとめたい」という必要性から、変数と配列の動機づけを行う | 数学の文字とは違い「後から書き換わる」点を意識させる |
| 20分 | 展開①（動画視聴） | 授業動画を視聴し、変数の概念（代入・書き換え）・配列の構造とインデックス（0から始まる）を学ぶ。トレース可視化ツールによる値の変化（`a = a + 1` など）も動画で確認する | 視聴中はワークシートの空欄・トレース欄を埋めながら見るよう指示する |
| 15分 | 展開②（Predict→Run） | 変数の入れ替えや配列要素の参照について、出力をペアで予想（Predict）→実行して確認（Run）する。トレース欄に各行の変数の値を書き込む | 「実行前に変数の値を書く」トレースの型を体験させる。範囲外インデックスのエラーも見せる |
| 7分 | まとめ | 変数・配列・インデックスを板書整理する。次回からPRIMの演習に入り、コードを「読んで・直す」段階に進むことを予告する | トレースが今後の演習の土台になることを伝える |

## 板書・スライド構成

1. 今日の目標
2. 変数：値の入れもの（代入と書き換え）
3. `a = a + 1` をトレースで追う
4. 配列：値を並べる（インデックスは0から）
5. トレースの書き方（各行で変数の値を記録）
6. まとめ：次回からPRIM演習へ
