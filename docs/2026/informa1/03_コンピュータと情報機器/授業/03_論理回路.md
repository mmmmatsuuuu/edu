# 第3時：論理回路

## 本時の目標

AND・OR・NOT回路の仕組みを理解し、真理値表を正しく読み書きできる。組み合わせ回路の概念を理解し、半加算回路の構成と動作を説明できる。

## キーワード

| 用語 | 説明 |
|---|---|
| 論理回路 | 0と1の信号を処理する電子回路。コンピューターの演算の基本単位 |
| AND回路（論理積） | 2つの入力がどちらも1のときだけ1を出力する回路 |
| OR回路（論理和） | 2つの入力のどちらか一方でも1であれば1を出力する回路 |
| NOT回路（否定） | 入力の0と1を反転して出力する回路 |
| 真理値表 | 入力の組み合わせすべてに対して出力がどうなるかを一覧にした表 |
| 組み合わせ回路 | 複数の基本論理回路を組み合わせて構成した回路 |
| 半加算回路 | AND回路とXOR回路を組み合わせて1ビット同士の加算（和と桁上がり）を行う回路 |

## 準備物・教材

- 教科書：情報Ⅰ Flex 第3章
- 授業動画（AND/OR/NOT/XOR回路・真理値表・組み合わせ回路・半加算回路）
- スライド（各回路の記号・半加算回路の回路図）
- 練習プリント（真理値表の穴埋め・組み合わせ回路の出力計算・半加算回路の確認）

## 授業展開

| 時間 | 段階 | 学習活動 | 指導上の留意点 |
|---|---|---|---|
| 8分 | 導入 | 「コンピューターは0と1しか扱えない。どうやって計算しているのか？」と問いかけ、演算の最小単位として論理回路を位置づける | 電気の「オン/オフ」と0/1の対応を確認してから入る |
| 20分 | 展開①（動画視聴） | 授業動画を視聴し、AND・OR・NOT・XOR回路の記号と真理値表、組み合わせ回路の概念、半加算回路（XOR＋AND）の構成と1ビット加算の仕組みを学ぶ | 視聴中は各回路の真理値表の空欄をワークシートに埋めながら見るよう指示する |
| 15分 | 展開②（活動） | 練習プリントに取り組む（①各回路の真理値表の穴埋め・②組み合わせ回路の出力計算・③半加算回路の真理値表の確認）。完成後、全体で答え合わせをする | 真理値表は「全入力パターンを必ず書く」習慣をつけさせる。「1＋1＝10（2進数）」の桁上がりがAND出力に対応することを答え合わせで確認する |
| 7分 | まとめ | 各回路のポイントをキーワードで板書整理する。次時（情報の内部表現）との接続として「実際の数はどう0/1で表すのか？」と問いかける | 計算が苦手な生徒が多い場合は答え合わせの1問を全体で一緒に解く |

## 板書・スライド構成

1. 今日の目標
2. 論理回路とは（0と1を扱う電子回路）
3. AND回路：回路記号・真理値表
4. OR回路：回路記号・真理値表
5. NOT回路：回路記号・真理値表
6. 組み合わせ回路の例①：AND回路（1種類の回路のみで構成される最も単純な例）
7. 組み合わせ回路の例②：AND＋OR回路（2種類を組み合わせた例・回路図と真理値表）
8. 組み合わせ回路の例③：XOR回路（排他的論理和・回路図と真理値表・「どちらか一方だけ1のとき1」の直感的な説明）
9. 半加算回路の回路図と真理値表（和・桁上がり）
10. まとめ：各回路のポイント整理
