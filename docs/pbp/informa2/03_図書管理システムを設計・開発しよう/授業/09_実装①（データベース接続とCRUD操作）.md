# 第9時：実装①（データベース接続とCRUD操作）

## 本時の目標

1. Pythonのsqlite3モジュールを使ってデータベースに接続・切断できる
2. PythonからSQL（CREATE・INSERT・SELECT）を実行し、結果を取得できる
3. BookクラスとデータベースのCRUD処理を組み合わせたBookRepositoryクラスを実装できる

## 取り扱う学習指導要領の内容

- エ-(イ) データベースの設計と活用
- エ-(ウ) プログラミング

## 準備物・教材

- 教師：DB接続実装Colabノートブック（雛形）
- 生徒：PC、これまでのクラス実装とSQLファイル
