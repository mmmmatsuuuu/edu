# 単元名：テクノロジーで社会課題を解決する探究

## 単元の目標

1. DX・スマートシティ・フィンテックなどの情報技術の社会応用事例を多角的に理解できる
2. デジタルデバイドの原因と影響を分析し、情報格差の解消に向けた考えを表現できる
3. GDPRや個人情報保護法・AIガイドラインなど、情報技術に関わる法・倫理の枠組みを理解できる
4. 関心を持つ社会課題に対して情報技術の貢献可能性を探究し、提案としてまとめ発表できる

## 授業時数

全6時間

## 取り扱う学習指導要領の内容

- ア-(ア) 情報技術の発展と社会の変化
- ア-(イ) 情報に関する法規・倫理・セキュリティ
- イ-(ア) メディアとコミュニケーション手段の特性
- オ 情報と情報技術を活用した問題発見・解決の探究

## 単元の評価基準

| 観点 | 評価基準 | 評価方法 |
|---|---|---|
| 知識・技能 | DX・デジタルデバイド・情報技術関連法の内容を正確に説明できる | 小テスト・振り返りシート |
| 思考・判断・表現 | 社会課題と情報技術の関係を多角的に考察し、説得力ある提案を作成できる | 探究まとめ・発表 |
| 主体的に学習に取り組む態度 | 自分が関心を持つ社会課題を主体的に選び、粘り強く探究活動に取り組んでいる | 観察・進捗記録 |

## 評価方法

| 評価方法 | タイミング | 備考 |
|---|---|---|
| 振り返りシート | 毎時間 | 自己評価と気づきの記録 |
| 探究まとめシート | 第5時提出 | 課題設定・調査・提案の流れ |
| 探究発表 | 第6時 | ルーブリックで評価 |
