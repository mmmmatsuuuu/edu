# 第3時：データ前処理（欠損値・外れ値・正規化）

## 本時の目標

1. pandasを使って欠損値を検出し、適切な方法（削除・補完）で処理できる
2. 外れ値の検出方法（IQR法・Z得点法）を理解し、対処の判断ができる
3. Min-Max正規化と標準化の違いを理解し、Pythonで実装できる

## 取り扱う学習指導要領の内容

- ウ-(ア) データの収集・整理・分析
- エ-(ウ) プログラミング

- 教師：前処理演習用Colabノートブック（欠損値・外れ値を含む練習データ付き）
- 生徒：PC、Google Colab、前時収集のデータ
