# 第6時：信頼区間の概念

## 本時の目標

1. 標本分布と標準誤差の概念を理解し、点推定と区間推定の違いを説明できる
2. 95%信頼区間の意味（「100回試行したとき95回はこの区間に母平均が含まれる」）を正しく説明できる
3. Pythonでscipy.statsを使って平均の信頼区間を計算し、グラフに誤差棒として表示できる

## 取り扱う学習指導要領の内容

- ウ-(イ) 統計的推測
- エ-(ウ) プログラミング
