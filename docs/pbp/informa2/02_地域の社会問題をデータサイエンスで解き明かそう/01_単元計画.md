# 単元名：地域の社会問題をデータサイエンスで解き明かそう

## 単元の目標

1. 公開データを収集・前処理し、分析可能な形に整備するスキルを習得する
2. 記述統計・統計的推測（仮説検定・信頼区間）・回帰分析の概念と手法を理解して適用できる
3. matplotlibを使ってデータを適切なグラフで可視化し、発見した知見を伝える資料を作成できる
4. データ分析に基づく政策提案を論理的にまとめ、プレゼンテーションで表現できる

## 授業時数

全13時間

## 取り扱う学習指導要領の内容

- ウ-(ア) データの収集・整理・分析
- ウ-(イ) 統計的推測
- ウ-(ウ) データの分析・活用（機械学習含む）
- エ-(ウ) プログラミング
- オ 情報と情報技術を活用した問題発見・解決の探究
